# RTranslate
> Translate any word or sentence in real time on your screen with OCR and DeepL API. Primarily for reading documents, articles, novels/visual novels. Unfortunately, only EN to TH is available as of now because I initially intended this to be used for personal purposes.

## Download (No Installation Required)
You do not need to install Python or use a terminal to run this app. 

1. Go to the **[Releases](../../releases)** tab on the right side of this GitHub page.
2. Download the latest `RTranslate_vX.X.zip` file.
3. Extract the `.zip` file anywhere on your computer (like your Desktop).

## First Time Setup (API Key)
Because this app relies on DeepL's translation servers, you will need your own API key.

1. [Get an API key from DeepL](https://www.deepl.com/en/pro/change-plan#api) (The free tier is perfect for personal use).
2. Open the extracted `RTranslate` folder and double-click **`RTranslate.exe`**.
3. A popup will appear letting you know it generated a file called `api_key.txt`. 
4. Open `api_key.txt` with Notepad, paste your DeepL API key inside, and save it.
5. Run **`RTranslate.exe`** again. 

*(Note: If your key is correct, you will get a welcome popup confirming the app is running in the background).*

## How to Use
Once the app is running quietly in the background, you can use these hotkeys from anywhere in Windows:

* `Ctrl + Alt + T`: Freeze the screen and draw a translation box. (Press again while open to start a new snip).
* `Esc`: Close the translation popup and return to Windows.
* `Ctrl + Alt + Q`: Completely shut down the application running in the background.
